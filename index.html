📢 Canais de Destino

Adicione os canais onde o bot vai enviar os sinais.
Use o @username do canal (ex: @meucanal) ou o Chat ID numérico (ex: -1001234567890).

❓ Como descobrir o Chat ID
Opção 1 — Usar @username (mais fácil)
Se o canal for público, use diretamente o @username
Ex: seu canal é t.me/palpitaio → use @palpitaio

Opção 2 — Chat ID numérico (canais privados)
1. Encaminhe qualquer mensagem do canal para @userinfobot
2. Ele responde com o Forwarded from chat e o ID
3. O ID começa com -100 para canais

⚠️ Lembre-se
O bot precisa ser Administrador do canal para enviar mensagens.
🤖 Não tem um Bot ainda?
1. Abra o Telegram e busque @BotFather
2. Envie /newbot
3. Escolha um nome e um @username para o bot
4. O BotFather envia o TOKEN — copie e cole acima
5. Adicione o bot como Administrador no seu canal
